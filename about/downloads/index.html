Downloads

*All of the data presented in COVIDmapper is also available for download through the project's GitHub repository.

Documentation

The downloadable data is presented in tab-separated files and organized at different geographic levels using a folder hierarchy: (1) global level, (2) continent level, (3) country level, (4) province/state level, and (5) county level. Both actual data and projected data are available for download.


Actual data is presented in two main forms:

  1. Daily summaries for all sub-regions with dates prior to today's date (<region name>_<date>.txt)
  2. Regional summaries for all past dates compiled in a single file (accumulated.txt)

Projected data is presented in two main forms:

  1. Daily summaries for all sub-regions with dates after and including today's date (<region name>_<date>.txt)
  2. Regional summaries for today and future dates compiled in a single file (predicted.tsv)


Note: IFR (in files for actual data) stands for infection fatality rate and represents the proportion of deaths among all the infected individuals (taking into account asymptomatic and undiagnosed infections).
